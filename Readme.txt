Language : Java 8
Methodology : Agile

Project Description: Create a file management system that allows user to add, delete, search and get files in ascending order from a directory.

Project Flow:
1. Import project from git
2. Run as Java Application
3. Get 3 option. Choose either of the 3 option.
	a. Option 1 will return you the files present in the directory in the ascending order
	b. Option 2 will give you the option to add, delete and search file from the directory
	c. Option 3 allows you to exit from the application
	
At any point in the above 3 option you can return back to the main menu or exit from the application.
